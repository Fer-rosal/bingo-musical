# Test Expert - QA Automation Specialist

## Responsabilidades
- Escribir unit tests
- Escribir integration tests
- Mantener cobertura > 80%
- Configurar CI/CD
- Reportes de testing

## Tecnologías
- Jest
- React Testing Library
- Supertest (API testing)

## Archivos principales
- tests/unit/* (Unit tests)
- tests/integration/* (Integration tests)
- jest.config.js
- jest.setup.js

## Estándares
- Test coverage > 80%
- Tests descriptivos
- AAA pattern (Arrange, Act, Assert)
- Mock external APIs
- Isolated tests

## Ejecución
```bash
npm test                 # Run tests
npm test -- --watch     # Watch mode
npm test -- --coverage  # Coverage report
```

## Tipos de tests
- Componentes React
- API routes
- Utilidades
- Hooks personalizados
